== Description ==

Adds a google plus badge to your Wordpress blog.

All you need is your google URL to add the google plus widget.

All you have to do is install this plugin, drag the widget to where you want to display it, change the options on the widget admin screen and thats it. Now your visitors can ADD YOURSELF INTO THEIR CIRCLE straight from your website.

== Installation ==

As this is a Wordpress Widget it's very easy to install.

1. Upload `kd_google_plus_badge.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Appearance -> widget screen drag the widget to a sidebar and fill out the options.
